hyperAce: Performs the adaptive cosin/coherent estimator algorithm using covariance matrix
hyperAceR: Performs the adaptive cosin/coherent estimator algorithm using correlation matrix
hyperAceR_M: Performs the adaptive cosin/coherent estimator algorithm MOVING ACE with a moving subset of pixels for correlation estimation
hyperAceR_RT: Real-time adaptation using pseudoinverse and correlation matrix updating 
hyperAceR_RTSM: Real-time adaptation using pseudoinverse and correlation matrix updating via Sherman-Morrison formula
hyperAceR_SUB: Performs the adaptive cosin/coherent estimator algorithm with correlation matrix using subset of rows/columns
hyperAceR_SUBRAND: Performs the adaptive cosin/coherent estimator algorithm with correlation matrix using randomly selected subset of rows/columns


hyperAceR_FP: Fixed point testing
hyperAceR_RTA: Real-time adaptation using pseudoinverse and correlation matrix updating, with exclusion of target pixels (not yet finished)
